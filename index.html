整体策略

01

🌐 媒体生态

以 芒果TV 品牌力撬动海外直媒合作伙伴和国际 SSP 加入广告联盟，构建全球化媒体网络。

02

💰 预算来源

以 优质视频广告联盟 吸引出海及海外品牌广告预算，形成品牌-媒体-用户正循环。

03

🚀 发展路径

以视频广告联盟为 内核，拓展多元化广告媒体联盟，承接更多效果广告预算。

📦

产品体系

📌 发展策略

  • 起始 海外视频广告联盟 — 以芒果TV国际版为核心，接入亚洲、拉美、中东及欧美AVOD平台流量
  • 拓展 海外效果广告联盟 — 开拓视频流量以外的多元化媒体，从主流效果广告对媒体属性的需求逐步接入更多短剧、休闲游戏、工具等 APP流量

📺 媒体类型

  • O2O 芒果TV国际版 — 核心直营
  • AVOD 第三方视频/AVOD直媒，如 TrueID
  • SSP 国际SSP，如 OpenX

📈 广告形式类型

  • 🎬 视频前贴片 / 中插贴片
  • 📱 开屏 / 插屏 / 横幅
  • 🔄 信息流 / 激励视频

📱 终端类型

  • 📱 APP 应用
  • 🌐 WEB
  • 📺 CTV
  • 💻 其他智能硬件

💵 预算类型

  • PMP 品牌 PMP，如 TTD
  • ORTB OpenRTB 公开竞价，如 Google OA

🔄 增长飞轮

  • 更多媒体 → 更多流量 → 更具吸引力的预算池
  • 更多预算 → 更高售卖率 → 媒体收益提升 → 吸引更多媒体
